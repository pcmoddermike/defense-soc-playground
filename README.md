# Defense SOC Playground

## Objectives

- Translate attacker techniques into actionable detection logic
- Analyze false positives and tuning opportunities
- Build small Python utilities that support SOC workflows

## Repository Structure

## Detection Philosophy
- Detections prioritize signal quality over alert volume
- Assumptions and log dependencies are documented
- Each detection aligns to MITRE ATT&CK techniques
- Queries are written to be understandable and explainable
